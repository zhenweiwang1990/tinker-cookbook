# Components

## TrainingList
左侧训练历史列表组件。显示所有训练会话，支持选择。

## TimelineList
中间 Timeline 列表组件。显示选中训练的所有 baseline、step 和 eval，按时序排列。

## DetailPanel
右侧详情面板组件。显示选中 timeline 项的详细信息，包括：
- Summary（摘要信息）
- Task（任务信息）
- Validation（验证结果）
- Rollouts（滚动列表）

## RolloutList
Rollout 列表组件。可折叠显示所有 rollout，点击可查看详情。

## RolloutDetail
Rollout 详情组件。显示完整的 rollout 信息，包括所有 turns、actions 和 observations。
